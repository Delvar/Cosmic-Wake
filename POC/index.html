POC Versions

These are all very early examples of the Cosmic Wake project, where a lot of the initial decisions were made and tested. It's amazing to see how far it's come!

Key Action
Arrow Keys Fly the ship around
T Select targets (planets, jumpgates, etc.)
J Jump through jumpgates (may have timeout to prevent rapid jumping)
L Land on targeted planet (fly over slowly first) or take off

Note: Not all versions have the same features.
